# tuxivoice
This Is a tuxi voice Script
## Dependencies
tuxi, dmenu, GNU grep, cat, festival with a voice pack(festival festival-us festival-english need to be downloaded for arch based distros)
## Installation
Install the dependencies and make the files tuxi.txt and file.txt in your home directory. Make Sure it is empty
Then git clone the repo and put tuxivoice in /usr/bin.
That is it
